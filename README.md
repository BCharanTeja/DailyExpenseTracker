# Daily Expense Tracker

A Kotlin Android application for tracking daily expenses and monthly spending.

## Features
- Add daily expenses
- View expense history
- Show today's date
- Calculate monthly expense total
- Clear all expenses
- Dynamic UI updates

## Tech Stack
- Kotlin
- Android Studio
- XML Layouts
- OOP Concepts
- Collections (ArrayList)
- Git & GitHub

## Architecture
UI (XML)
↓
MainActivity
↓
Expense Data Class
↓
ArrayList
↓
Expense History & Total Calculation

## Future Enhancements
- RecyclerView
